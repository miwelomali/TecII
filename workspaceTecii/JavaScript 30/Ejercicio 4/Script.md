# Script Ejercicio 12 By Dino.

Lo único que se aprendió en este ejercicio es la manera de obtener los keystrokes y como relacionar el conjunto de keystrokes con la palabra secreta.

Al ejercicio se le agrego lo siguiente:

Palabra secreta: kirby
Un gif de kirby comiendo ramen.
Un botón con un evento click, el cual reproduce el audio de poyo cada vez que se da click.
Una canción que inicia en segundos diferentes, para ser especifico entre (0 y 34 segundos), ya que el numero 34 se acerca por 4 tiempos al drop de la canción.
Un párrafo que muestra todo lo que se esta escribiendo en la pagina.

También se aprendió que el código konami no funciona en todas las paginas web que dicen tenerlo (sad face).
